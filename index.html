  1. DESCRIPCION

Discusion de conceptos de diseño y desarrollo de paginas Web. Aplicacion de los lenguajes HTML y JavaScript conjuntamente con CSS para la creacion de paginas Web. Creacion y adaptación de elementos de multimedios para ser utilizados en paginas interactivas. Requiere 45 horas de conferencia-laboratorio.3 creditos

  1. OBJETIVOS

Se espera que al finalizar el curso, el estudiante pueda:

  1. Discutir conceptos de diseno y desarrollo de páginas Web.
  2. Aplicar los lenguajes HTML y JavaScript conjuntamente con CSS para la creación de páginas Web.
  3. Crear y adaptar elementos multimedios para ser utilizadas en paginas interactivas.
  4. Entender las implicaciones éticas, sociales y legales inherentes a la disciplina.
  1. CONTENIDO TEMÁTICO
    1. Conocer los editores del lenguaje HTML.
      1. Historia de los editores de HTML
      2. Introducción a HTML.
        1. Listas numeradas, no numeradas y de definición
        2. Enlaces internos y externos
        3. Incorporación detablas, imágenes y videos
        4. Diseño de formularios e incorporación de “meta elements”
    1. Creación de páginas interactivas.
      1. Creación de páginas interactivas utilizando un editor de HTML.
      2. Publicación de páginas interactivas.
    1. Introducción a CSS
      1. Uso de CSS.
      2. Reglas de CSS.
      3. Incorporación de CSS a una página interactiva.
    1. Introducción a JavaScript
      1. Definición.
      2. Incorporación dev un javascipt a páginas interactivas.
      3. Integración de elementos visuales que utilizan animación vectorial.
